# Skills

My OpenClaw Skills collection.

## pm-self-check

AI 时代产品决策自查框架。帮助产品经理在行动前，用「产品自查 + 产品自问」两层结构确认方向是否正确。

**适用场景：** 新产品立项、AI 产品评估、产品方向复盘、体验恶化排查

## 安装方式

### 让你的 OpenClaw 帮你装

复制这个页面链接，发给你的 OpenClaw agent，说「帮我安装这个 Skill」。

### 从 GitHub 安装（手动）

```bash
git clone https://github.com/Vikkyday1beginner/Skills.git
cp -r Skills/pm-self-check ~/.openclaw/workspace/skills/
```

### 下载 ZIP

点右上角 **Code → Download ZIP**，解压后将 `pm-self-check` 文件夹放到 `~/.openclaw/workspace/skills/` 目录下。

## 使用方式

安装后，重启 OpenClaw，直接告诉你的 agent：

- "帮我评估这个产品方向"
- "这个功能值得做吗"
- "帮我做一个产品自查"

agent 会自动引导你完成自查流程。
